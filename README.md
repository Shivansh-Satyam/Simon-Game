# Simon-Game
A simple web-based Simon Says memory game built with HTML, CSS, and plain JavaScript

<h2>How to Play</h2>
<ul>
<li>Press any key to start the game.</li>
<li>The game will flash a color sequence.</li>
<li>You must click the buttons in the same order.</li>
<li>If you are correct, the game moves to the next level and adds one more color to the sequence.</li>
<li>If you make a mistake, the game is over and your score (the level you reached) will be displayed.</li>
<li>To restart, just press any key again.</li>
</ul>

<h2>How to Run</h2>
<ul>
  <li>Place all the files (index.html, style.css, and app.js) in the same folder.</li>
  <li>Open the index.html file in your web browser.</li>
  <li>The game will load and be ready to play.</li>
</ul>
